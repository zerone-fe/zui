# zui

## 编译组件库

```bash
yarn lib
yarn release
```

## 安装

```
yarn add @zerone-fe/zui
```

## Project setup

```
yarn install
```

### Compiles and hot-reloads for development

```
yarn serve
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```
